# How to write a good README

## How to write a good README for your GitHub project?

- 一个好的 README 文件必须要有以下元素：

  - 大标题和内部标题
  - Introduction；对项目的介绍
  - 技术介绍
  - 如何运行

  此外，以下元素也是加分项：

  - 目录
  - 插图
  - 功能范围
  - 使用用例
  - 项目状态
  - Sources
  - Other information
